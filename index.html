M5DialBMS

Battery Management Monitor · Tesla & BMW Module Support

1

What you need

  • M5Stack Dial with a USB-C cable that supports data transfer
    (many cheap cables are charge-only — if it doesn't connect, try a different cable)
  • Google Chrome or Microsoft Edge browser
    (Firefox and Safari are not supported by the Web Serial API)
  • A CAN transceiver connected to Grove B (SN65HVD230 recommended)

✓ The M5Stack Dial does not require holding any buttons to enter flash mode — it connects automatically when you click Install.

Install Firmware

This will erase the device and install the latest M5DialBMS firmware.
The process takes about 30–60 seconds.

Click to Install M5DialBMS
⚠️ Your browser doesn't support Web Serial.
Please use Chrome or Edge.
Alternatively, flash manually using esptool-js at address 0x0.
⚠️ This will erase all existing data on the device. If you have custom settings saved, note them down first.
3

After flashing

  1. The M5Dial reboots and shows the BMS startup screen
  2. Connect to WiFi network M5DialBMS — password M5DialBMS
  3. Open http://192.168.4.1 in your browser
  4. Go to the Settings tab and configure your pack: select CMU type, set number of series modules, cells per module, and voltage thresholds
  5. Connect your CMU modules — cell data will appear on the display and dashboard within a few seconds
💡 WiFi is off by default at boot. Enable it via the Settings tab (WIFI=1) or with a long press of the encoder button at any time.

CMU Types Supported

  • ✓ Tesla BQ76PL536A UART
  • ✓ BMW i3 CSC Standard
  • ✓ BMW i3 Bus Pack (Vicinity)
  • ✓ BMW Mini-E CSC
  • ✓ BMW PHEV SP06/SP41

Features

  • ✓ SimpBMS CAN TX (zombieverter)
  • ✓ Web dashboard at 192.168.4.1
  • ✓ SavvyCAN / GVRET on port 23
  • ✓ Active cell balancing (Tesla)
  • ✓ 240×240 round LVGL display
